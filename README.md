# ATR
Automatic tuning robot for deep learning
## Features
- [x] Auto kill process
- [x] Auto start process
- [x] Listener for process crash
- [x] GPU load reading
- [x] CPU load reading
- [x] Memory usage reading
- [x] Auto assign GPUs
- [x] Auto tune hyperparameters
- [x] Auto read training results
